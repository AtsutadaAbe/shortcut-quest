# Shortcut Quest

Excelショートカットをゲーム感覚で学べるWindows向け学習ゲームです。

問題に対応するショートカットキーを実際に押して、敵を倒しながら学習できます。

---

## 特徴

- 通常戦（ランダム出題）
- ボス戦あり（5問連続）
- クリアタイム計測
- ランク表示（勇者・剣士・冒険者・旅人）
- スライム画像ランダム表示
- 効果音あり

---

## 動作環境

- Windows

---

## 遊び方

1. ゲームを起動
2. スタートボタンを押す
3. 表示された問題に対応するショートカットキーを押す
4. 敵を倒してクリアを目指す
